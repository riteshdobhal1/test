About Section Views

The Section View allows a support user to view, organize, filter and group non-event sections within a log bundle. Sections thus organized can be saved for quick retrieval.

Support users look at multiple non-event sections within a log file, while trying to troubleshoot an issue. Within the log files, these sections:

  • Are spread across the logs and therefore, the user has to scroll up and down the log to cross reference data across logs. It would help if all relevant sections were next to each other every time
  • Have data in rows or columns that are not relevant for the problem at hand. Looking at data that is not relevant to the problem slows down the troubleshooting process. It would help if users are allowed to filter each sectionâ€™s data ( both rows and columns) so that they can only look at data relevant to that problem
  • Are not represented in an easy-to-analyze way. Some sections are better visualized when the data is transposed. It would help if each section could be transposed and viewed.

Section View is a powerful tool that allows users to view sections of interest either as table or as text and also see difference between multiple versions of a set of sections. In Table view, you can filter section data by rows or columns, transpose views, and group and order them in a way that is meaningful to a specific problem, and more importantly, to save the view. This saved view can then be applied to any log bundle from any device, to help the support teams to quickly view the relevant parts of the log represented in a format that enables quicker troubleshooting.

Section Views allow you to view the sections of the log file, observe the configuration differences in log bundles, view each log section in a tabular view, and apply transformations to the tabular view. Apart from that, you can download the raw log, export transformed sections to pdf, and save the customized views of the log sections.

The high level capabilities of the Section View include:

  • Select and view specific portions of a log file
  • Apply filters to suppress the columns of rows of data within that section
  • View selected sections as raw log file ( text format)
  • View difference for a given set of sections across multiple log bundles received over time
  • Download the raw bundle
  • Save a customized view of the log view and apply again whenever required
  • Transpose a section from the row column and vice versa
  • Share the saved views with others in your company/department
  • Export section views to PDF or Excel
