Users List

To know more about users, click here.


Number

Description

1

Click here to expand the filter.  This filter helps in performing a specific search based on the following listed parameters.  You can select/unselect the values of these parameters to search the users list.

  1. Email
  2. Role
  3. System Group
  4. Created On
  5. Modified On
  6. Last Login
  7. Status
  8. User Type

This filter works in conjunction with the search annotated with 3.

Example:





  1. Enter the search criteria to list the email addresses.
  2. Select one or more email addresses to view a filtered list of users having the selected email addresses.
  3. Displays the number of values selected in the search list.  
  4. Click here to expand/collapse the filter parameters.
  5. Close the filter pane to see .  Click on to clear all the filter selections.

2

Select from the following options:

  • All
  • Internal 
  • External

All – Lists all the users – both Internal and External.

Internal – Lists only internal users.

External – Lists only external users.

Note:  On selecting Internal or External, the corresponding option is auto selected in the filter (annotated as 1) for the parameter Role Type.

3

Click here to know more about bulk edit.

4

Enter search criteria to search by the following parameters:

  • User name (First name & Last name)
  • Email
  • Role name
  • System Group
  • Status
  • User Type

5

Click here to download the users list in the .csv (comma separated value) format. No filters (that are applied on the screen) are applicable to the list that is downloaded. Always the entire list of end customers will be downloaded.

6

List of existing users with the following headers:

  • Name
  • Email
  • Role
  • System Group
  • Created On
  • Modified On
  • Last Login
  • Status
  • User Type
  • Actions

7

Select a suitable page size.  The available page sizes are 10,20,30, and 50.  

By default, the page size is set to ten.  Hence, by default, the number of users shown per page is ten.

8

Use pagination to navigate between the pages of users list.  Click on page numbers to navigate across pages of users list.  Number of pages displayed depends on:

  • Total number of roles in the list
  • Page Size

9

Click here to delete single/multiple selected users.  

Delete option is enabled only when one or more users are selected in the list.

Note:  

All users except the user that is currently logged in can be selected for deletion.
