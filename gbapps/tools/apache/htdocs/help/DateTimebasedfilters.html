Date/Time based filters



By default, Logvault displays historical data up to a limit of one year.  Hence, From date is set accordingly and To date is set to current date.  If your organization wishes to retain/display historical data for more than a year, contact support@glassbeam.com.

Quick Time Filter has the following options.



By default, the filter criterion is Custom Date Range.

Note:  

With Custom Date Range as the filter criterion, From and To date options are available for selection.

The search button is active only for Custom Date Range.

Click here to select a date from the date picker.  The From date can be any date prior to To date.   You cannot select From date that is older than one year from the current date.

Click here to select a date from the date picker.  By default, To date is set to current date.  You can select any date that is prior to current date and later than From date.

Click here to apply the From and To dates as the search criteria.  


Based on the date filters, the Data display and navigation section and the Graph timeline section display all the records meeting the filter criteria.  The filter criteria are applied across all the bundles in the Log Vault.
