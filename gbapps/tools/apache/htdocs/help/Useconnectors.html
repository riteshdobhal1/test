Use connectors



From the Connectors tab, you can connect to data housed in a cloud database or on a server in your enterprise. You need to supply connection information for each data connection that you make. For example, for most data connections, you need to supply a server name and your sign-in information.

**Google BigQuery requires OAuth when creating data sources from the web. Learn more about how server administrators can Set up OAuth for Google.

Supported Connectors has information on how to connect Workbench to each of these connector types to set up your data source. 

When Workbench successfully connects to your data, the Data Source page opens so that you can prepare the data for analysis and begin building your view. To learn more, see Creators: Prepare Data on the Web.
