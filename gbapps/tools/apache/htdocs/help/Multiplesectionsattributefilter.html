Description

List of sections selected for comparison of attributes.  At least two sections in a log have to be selected for comparison.

Example: The screen shot above shows SCANNER INFO and tubeUsage/Scan Information sections for comparison.

List of all the different attributes that are common across all the different sections that have been selected.

Example:  The screen shot above shows that Facility, Department, Device Category etc., are attributes that are common in SCANNER INFO and tubeUsage/Scan Information sections.  You can scroll down the window to view the entire list of common attributes.

Enter search criteria for any common attribute.  All rows from the selected sections that have the common attribute match the entered search criteria will be displayed.

Example:  Follow the steps below to search any specific value for Facility attribute:

  1. Enter HMC as search criteria for Facility attribute.
  2. Click on Done.

The table view of the selected sections displays all the rows of the selected sections for SCANNER INFO and tubeUsage/Scan Information sections.

Note:  You can enter search criteria for multiple attributes.  In this case, all search criteria on the respective attributes are applied in conjunction to display the rows that match all the criteria entered.
