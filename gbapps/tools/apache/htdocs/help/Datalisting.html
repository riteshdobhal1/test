Click here to expand the menu options as shown in the screen shot below.


Show Original time:  Selecting this allows the user to see the creation time of the log bundle - in the original time zone where it was created. By default Glassbeam converts all time zones to UTC time zone –for storage and display.


Show:  Click on the drop down list of numbers.  The numbers represent the number of line items per page in the data list that will be displayed.

Example:  select 20 from the drop down list.  The data list will now display 20 line items per page in the data list.  Hence, If there are 100 line items in the data list, there will be 5 pages with 20 line items each.


Download:  Select one or more line items in the data list (annotated by 8) and click on to download the selected log bundles.
