Column Filter

Click on to expand the column filter as follows.


Number

Description

1

Type:  Click here to expand or contract the filter list.  2 pre-determined types are listed.

  • Internal
  • User Created

Example:  

Internal(6) means that there are 6 internal dashboards.

Note:  When Internal or User Created options are selected in the Global Filter, the corresponding filter is selected in the column filter also.

2

Last Modified:    Click here to expand the following pre-determined list of Last Modified options.

Only one option can be selected at any point in time.  

Example:


Select Last Week and the same is displayed along-side the Last Modified filter parameter.

3

Tags:  This parameter is displayed in the column filter only if any dashboard has been tagged.  Otherwise, the parameter does not appear in the column filter.  On expanding the parameter, all the tags that have been assigned to the dashboards are listed.  Multi-select option is available.

4

Click here to clear all the filters selected.

5

Click here to close the column filter.  

Note:  

  • The following filter parameters have multiple select option:
    • Owner
    • Datasource
    • Tags
  • Datasource selection is disabled if Type is selected as Internal.  This is because, datasources are used only while creating user created dashboards.  You can select multiple datasources from the list.

  • The column filter always shows the number of filters selected in the column filter.

The four selections being 

    • Type (1 selection) 
    • Owner (2 selections – multi-select)
    • Datasource (1 selection)
    • Last Modified (1 selection)
