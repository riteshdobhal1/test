Local dashboard actions



Number

Description

1

By default, the dashboards are private.  

  • - means private
  • - means public

Click on the private/public icons to toggle between them.

Public:  By making a dashboard public, you can expand the visibility of the dashboard to all the roles that are associated with the dashboard.

Private:  By making a dashboard private, you can restrict the visibility of the dashboard to the dashboard owner alone.

To know more about roles and permissions, click here.

2

Click here to view the roles that are assigned to the corresponding dashboard.

While you can assign roles to multiple dashboards through the global action, the dissociation of roles with the dashboards needs to be done at each dashboard level.  

Following is the screen shot of the list of roles associated with a dashboard.


  1. Click here to dissociate a role from the dashboard.

Note:  When the dashboard ownership changes from one user to another user, the new owner’s corresponding role is automatically assigned to the dashboard. 

3

A single dashboard may have multiple sheets of reports.  Click here to view the sheets of the dashboard in different tabs.  

Note: Show Tabs is enabled only if Show sheets as tabs is checked for the workbook in the workbench application.

4

Click here to delete the dashboard.

5

Click here to schedule report distribution through e-mail.  On click, the scheduling window pops up.  This is done in order to set the intervals for schedule report distribution.

Following is a screen shot of the report scheduler:

6

Click here to set the dashboard as a home dashboard.  The home dashboard serves as a summary report.  

Note:  If you have a summary report, then Home Report screen, by default, shows the summary report.  Otherwise, only All Dashboards option is displayed.

7

Click here to copy the dashboard link (URL) to the clipboard.  This link can be subsequently copied to any editor in your computer.

Note:  This option is available only for Internal Dashboards.

8

Click here to compose a new email with the dashboard link embedded in the body of the email.

Note:  This option is available only for Internal Dashboards.

9

By default, the user who created the dashboard is the owner of the dashboard.  Click here to open a dropdown list of all the users.  Select from the dropdown list - a user to whom the dashboard ownership has to be transferred.  

Internal Dashboard Owner List:  Lists the users in the following roles:

  • Super Admin
  • Dashboard Admin / Report Admin – i.e., the user has been set as a Dashboard Admin

User Created Dashboard Owner List:  Lists the users in the following roles:

  • Super Admin
  • Power user – i.e., the user has been set as a workbench admin.

Note:  When the dashboard ownership changes from one user to another user, the new owner’s corresponding role is automatically assigned to the dashboard.
