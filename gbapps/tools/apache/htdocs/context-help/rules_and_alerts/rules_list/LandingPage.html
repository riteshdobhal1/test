    Column_Filter

    Click here to Search and display rules list based on specific column values.

    The column filter lists all the columns that can be displayed in the rules list of the landing page page and the ability to filter on those columns.

    Check a column name – to include the column and its details in the rules list.

    Uncheck a column name – to remove the column and its details from the rule list.

    Expand a column name to filter rules list based on the data of that column.
