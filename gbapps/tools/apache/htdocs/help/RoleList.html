Role List

To know more about Roles, click here.


Number

Description

1

Click here to expand the filter.  This filter helps in performing a specific search based on the following listed parameters.  You can select/unselect the values of these parameters to search the roles list.

  • Role Name
  • Role Type 
  • Product Name
  • Features
  • Explorer Data Restriction

This filter works in conjunction with the search annotated as 2.

Example:


  1. Enter the search criteria to list the roles.
  2. Select one or more roles to view a filtered list of roles.
  3. Displays the number of values selected in the search list.  In this case, two values are selected.
  4. Click here to expand/collapse the filter parameters.
  5. Close the filter pane to see .  Click on to clear all the filter selections.

2

Enter search criteria to search by the following parameters:

  • Role Name
  • Role Type
  • Product Display Name: This is applicable for those customers where Glassbeam has been implemented for multiple product lines. A user can be given access to more than one product line. 
  • Features (Example:  Dashboards, LogVault etc...)

3

Click here to download the roles list in the .csv (comma separated value) format. No filters (that are applied on the screen) are applicable to the list that is downloaded. Always the entire list of roles will be downloaded.

4

List of existing roles with the following headers:

  • Role Name
  • Role Type
  • Product Display Name
  • Features

5

Select a suitable page size.  The available page sizes are 10,20,30, and 50.  

By default, the page size is set to ten.  Hence, by default, the number of roles shown per page is ten.

6

Use pagination to navigate between the pages of roles list.  Click on page numbers to navigate across pages of roles list.  Number of pages displayed depends on:

  • Total number of roles in the list
  • Page Size

7

Click here to delete single/multiple roles by selecting them.  

Delete option is enabled only when one or more roles are selected in the list.

If there are users (external) associated with roles selected for deletion, such roles are not deleted unless the corresponding users are deleted.  A message is displayed regarding this information.

Deleting a role means – deleting all the products associated with the role including the role.

Note:  You cannot delete the admin role that is pre-configured by the Glassbeam Professional Services.

8

Click here to add product(s) line to a selected role.  Product(s) can be added to only one selected role.

9

Click here to add a new role.  Click here to know more.

10

Click here to edit a particular role-product combination in order to include/exclude features for the respective product.  Click here to know more.

11

Click here to delete a single product from the corresponding role.

A product is not eligible for deletion under following circumstances:

  • Users are associated to the role-product combination
  • Product is the only product associated to the corresponding role

12

Click here to select all roles in the list in the current page.  To know more about select all, click here.
