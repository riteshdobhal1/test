Main Filter

Click here to filter reports/dashboards based on the following options.

1. Internal - Lists only the internal dashboards.

Internal Dashboards are analytics created by Glassbeam Professional Services based on the requirement provided by your organization. These dashboards have drill down properties to access deeper insights on specific properties of your install base.

Example:  Internal (32) means that there are 32 internal dashboards.

2. User Created - Lists only dashboards created by the Glassbeam application users.

User created dashboards are the ones created using the Workbench application.

Example:  User Created (27) means that there are 27 user created dashboards.


Owner:

Click here to filter reports/dashboards based on owners. By default, the user who creates the dashboard is the owner of the dashboard.  However, Super Admin or the dashboard owner can transfer the dashboard ownership to other users.

Internal Dashboard:  Ownership can be transferred to users in the following roles:

  • Super Admin
  • Dashboard Admin / Report Admin – i.e., the user who has been set as a Dashboard Admin

User Created Dashboard:  Ownership can be transferred to users in the following roles:

  • Super Admin
  • Power user – i.e., the user who has been set as a workbench admin.


Datasource:

Click here to filter reports/dashboards based on the datasource that was used to create it. Datasources are interfaces to database tables that are exposed by Glassbeam Professional Services for users to create their own reports. Internal Reports, which are created by Glassbeam Professional Services, will not be linked to any datasource as these are custom reports that could be created using multiple data sources.   Hence, datasource selection is disabled if Type is selected as Internal.  


Internal dashboards are custom reports created by Glassbeam Professional Serivces using multiple data sources.  Hence datasource selection in the filter is disabled if Type is selected as Internal.


Note:

The column filter always shows the number of filters selected in the column filter.
