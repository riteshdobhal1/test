If the Glassbeam Professional Services team sets only one dashboard as health check dashboard, then All Dashboards tab is not displayed.   If there is more than one dashboard that is set as health check dashboard, All Dashboards tab is displayed.


Number

Description

1

Select from the drop-down list of end customer group name.  

As the first option, the drop-down displays All.  On selecting All, all the dashboards that are associated to all end customer groups for your customer account are displayed.

The drop-down lists all the end customer groups (individually) that are created for the customer account.  On selecting an end customer group, only those dashboards that are associated to the selected end customer group are displayed.

2

Select from the drop-down list of all health check dashboard owners.

Select to display only those dashboards that are owned by the selected user.
