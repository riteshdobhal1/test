Facet filter



Number

Description

List of all the facets along with the count of facet values.

Example:   means that Serial Number is a facet with 46 unique values – across all the log bundles in the log vault.

On click of , the facet can be expanded to view the list of facet values.  Every facet has a search area.  On entering search criteria, only the facet values that meet the search criteria are listed.

Example:  


Enter CT5 as the search criteria in the Serial Number facet.  All facet values which have CT5 as part of their serial number are listed. 

Click on Select All to select all the facet values listed.  

Note:  Select All option appears only when a search criteria is entered to filter the facet values.
