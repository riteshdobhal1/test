Logic Field – What You Can Do

Logical operators

Logical operators

What the operator does

Example

OR

Logical OR operation that can be used when the rule has more than one condition. You can define rules such as condition1 or condition2, which means the rule will trigger when either or both conditions are true. 

sv.vname = "" OR sv.vnumber> 0

AND

Logical AND operation that can be used when the rule has more than one condition. You can define rules such as condition1 AND condition2, which means the rule will trigger only when both conditions are true. 

sv.vname = "" AND sv.vnumber> 0


Mathematical operators

Mathematical operators

What the operator does

Example

Plus (+), Minus (-), and Multiplication (*)

Use to perform addition, multiplication, and subtraction of values between two integer columns. 

Use only on integer value columns.

sv.vnumber1 + sv.vnumber2 = 100

/  (division)

Use to perform division of column values. 

Use only on integer value columns.

sv.vnumber1 / 2 = 100


Comparison Operators

Comparison Operators

What the operator does

Example

IS NULL

Check if the column is NULL (Empty). Use only on String value columns.

sv.vname1 IS NULL (true)

LIKE

Returns true if value in a column matches with a string. It can match an exact string or part of a string if enclosed within %%. Use only on String value columns.

sv.vnumber1 LIKE '%123%' 

NOT LIKE

Returns true if value in the column does not match the given string. It can compare against an exact string or part of a string if enclosed within %%.   Use only on String value columns.

sv.vnumber1 NOT LIKE '%123%' 

IS NOT NULL

Check if the column is NOT NULL (Empty). Use only on String value columns.

sv.vname2 IS NOT NULL (true)

<=,>= ,  = ,  < ,  >

‘=’ compares if the left hand side and right hand side is equal.

‘<’ checks if the left hand side is lesser than right hand side

‘>’ checks if the left hand side is greater than right hand side

<= checks if the left hand side is lesser than or equal to right hand side

>= checks if the left hand side is greater than or equal to right hand side

Use only on Integer value columns.

sv.vnumber>= 0 (0 or 1...)

<>

‘<>’ compares if the left hand side and right hand side is not equal 

Use only on Integer value columns.

sv.vnumber<> 0 (other than 0)

LOWER

Converts string values to lower case before performing an operation on it. Use only on String value columns.

LOWER( {sv.vname2} ) = "johndoe"

UPPER

Converts string values to upper case before performing an operation on it. Use only on String value columns.

UPPER( {sv.vname1} ) = "JOHNDOE"

Example table

----------------------------------------------------------

sv.vname1(String)       | sv.vname2(Integer)

----------------------------------------------------------

 2                                    | 3                              

-----------------------------------------------------------

CONVERTTOINT

Converts string value column to INT. Use only on string value columns. 

CONVERTTOINT({sv.vname1}) > 2

CONVERTTOSTRING

Converts integer values to string. Use only on Integer value columns. 

CONVERTTOSTRING({sv.vname2}) LIKE '%3%'

REGEX

Apply a Regular expression on a column’s value (a string) to extract portions of interest before applying other operations on it. Use only on String value columns.

REGEX({sv.string},"([V][\w]+[\d]+)") = "JohnDoe123"

CONVERTTOINT(REGEX({sv.integer},"([\d]+)")) = 123

COUNT without REGEX

Counts the number of times a condition occurs. The count can then be used with other operations. Can be used in combination with mathematical operators.

COUNT( {sv.vname1} = 2) >= 2 (true)

COUNT( {sv.vname1} = 2) > 2 (false)

COUNT( {sv.vname1} = 2) > 0 (true)
