Add/Edit User


To add a new user, click on Add User (highlighted in the above screen shot).  The following window pops up for keying in the details of the new user.



Click here to know more about Roles, Users and System Groups.


Number

Description

1

Check here to toggle between Internal/External.  By default the user is added as Internal unless you toggle here to add external user.

2

Select a role from the drop-down list.  

For external users, only external roles are listed.

For internal users, only internal roles are listed. 

3

The Default Product drop-down lists all the products for the selected role.  

By default, the first product in the list is selected as the default product.

The default product that is chosen for a user profile becomes the landing product when the user logs in.  

However, all products associated to the role will be accessible to the user upon logging in.

4

Select the System Group from the drop-down list.  .  Only system groups that are associated with the selected product will be displayed.   By default the first system in the list is selected.

Upon associating a System Group, the user will be able to look up dashboard data and subscribe to rule alerts for only those devices that are associated to the System Group.

If 'All' option is selected, the user must be able to look up data for all the devices of all the products that are associated with the selected role.

Note: 'All' option exists only for internal users.

Example:  

Product prod1 contains the following device ids – 

D1, D2, D3

Product prod2 contains the following device ids – 

D4, D5, D6, D7.

System Group EC1 is associated to prod1 – D2

System Group EC2 is associated to prod2 – D4, D6

System Group EC3 is associated to prod1 – D3

System Group EC4 is associated to prod2 – D5, D7

External role R1 is associated with prod1.

External role R2 is associated with prod1, prod2.

For an external user EU1, selecting R1 and prod1 results in the following listing of system groups.

  • EC1
  • EC3

For an external user EU2, selecting R2 and prod2 results in the following listing of system groups.

  • EC2
  • EC4

5


Check the option to send e-mail notification to the user.  The option is enabled when:

  • System Group changes are done to the user profile
  • Editing the user profile using Edit User option

System Group change impacts the user’s rule subscriptions.  Hence, e-mail notification can be sent to the user by checking this option in order for the user to take further action as desired.

6

Click here to save the new user.


Note:  

  • All information except Email can be edited/changed for a user.
  • You cannot edit user details for an INVITED user.  
  • To know more about INVITED users, click here.
