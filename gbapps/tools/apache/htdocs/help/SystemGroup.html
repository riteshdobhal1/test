System Group

System Group is a list of groups created to restrict users to certain device data through HealthCheck Application. 

Each System Group is associated to specific devices that concern them.  

To know more about System Groups, click here.

Select a suitable page size.  The available page sizes are 10,20,30, and 50.

6

Use pagination to navigate between the pages of System Groups list.  Click on page numbers to navigate across pages of System Groups list.  Number of pages displayed depends on:

  • Total number of System Groups in the list
  • Page Size

7

Click here to delete single/multiple system groups by selecting them.  

Delete option is enabled only when one or more system groups are selected in the list.

Note:  If there are users (external) associated with system groups selected for deletion, you cannot delete such system groups unless the corresponding users are dissociated.  

If a system group, say, EC1 is part of another system group, you cannot delete EC1 unless the system group containing EC1 is deleted/dissociated. 

A message is displayed in both the above cases.

Click here to edit an existing System Group.  The following parameters cannot be edited for an existing System Group.

  • System Group Name
  • Product Name

However, on editing a System Group, serial numbers can be added or removed from the access list.

Click here to delete the corresponding System Group.  

If there are users (external) associated with this System Group, you cannot delete unless the corresponding users are deleted.  

If another system group contains this system Group, you cannot delete unless the containing System Group is deleted/dissociated.

A message is displayed regarding this information.
